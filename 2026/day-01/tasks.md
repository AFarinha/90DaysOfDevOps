# Tasks - Day 01

Use these commands as a practical checklist for the day. Replace placeholders such as `<...>` before running them.

## Commands

| Command | What it does |
|---|---|
| `cd 2026/day-01` | Changes into the target directory before running the exercise commands. |
| `touch learning-plan.md` | Creates one or more empty files if they do not already exist. |
| `git status` | Shows changed, staged and untracked files in the repository. |
| `git add learning-plan.md notes.md tasks.md` | Stages the listed files for the next commit. |
| `git commit -m "Add day 01 learning plan"` | Creates a commit from staged changes with the provided message. |
