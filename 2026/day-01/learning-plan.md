# 90 Days DevOps Learning Plan

## Current Level

DevOps learner building practical foundations in Linux, Git, scripting, Docker and cloud operations.

## Why I Am Learning DevOps and Cloud

I want to understand how production systems are built, deployed, monitored and recovered when something fails.

## Goals for the Next 90 Days

- Build strong Linux troubleshooting habits.
- Create practical CI/CD and automation examples.
- Deploy and operate containerized applications with Docker and Kubernetes basics.

## Core Skills to Build

- Linux processes, services, logs and permissions.
- Git, GitHub workflows and collaboration.
- Shell scripting, Docker and cloud server deployment.

## Weekly Time Budget

- Weekdays: 1 to 2 hours per day.
- Weekends: 3 to 4 hours for revision and hands-on practice.

## Consistency Plan

Work in small daily blocks, document each exercise, commit progress regularly and revise weak topics during revision days.
