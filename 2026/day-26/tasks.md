# Tasks - Day 26

Use these commands as a practical checklist for the day. Replace placeholders such as `<...>` before running them.

## Commands

| Command | What it does |
|---|---|
| `gh --version` | Checks the installed GitHub CLI version. |
| `gh auth login` | Starts GitHub CLI authentication. |
| `gh auth status` | Shows the active GitHub CLI authentication state. |
| `gh repo create devops-gh-cli-practice --public --add-readme` | Creates a GitHub repository from the terminal. |
| `gh repo list` | Lists repositories for the authenticated account. |
| `gh repo view <owner>/<repo>` | Shows details for a GitHub repository. |
| `gh issue create --title "Test issue" --body "Created from gh"` | Creates a GitHub issue from the terminal. |
| `gh issue list` | Lists GitHub issues. |
| `gh pr create --fill` | Creates a pull request from the terminal. |
| `gh pr list` | Lists pull requests. |
| `gh run list` | Lists GitHub Actions workflow runs. |
| `gh workflow list` | Lists GitHub Actions workflows. |
