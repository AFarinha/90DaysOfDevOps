# Tasks - Day 16

Use these commands as a practical checklist for the day. Replace placeholders such as `<...>` before running them.

## Commands

| Command | What it does |
|---|---|
| `cd 2026/day-16` | Changes into the target directory before running the exercise commands. |
| `chmod +x hello.sh variables.sh greet.sh check_number.sh file_check.sh server_check.sh` | Changes file or directory permissions. |
| `./hello.sh` | Runs the first Bash script and prints the DevOps greeting. |
| `./variables.sh` | Runs the variables script and demonstrates variable expansion and quoting. |
| `./greet.sh` | Runs the interactive greeting script that asks for name and favourite tool. |
| `./check_number.sh` | Runs the number check script and classifies the input as positive, negative or zero. |
| `./file_check.sh` | Runs the file check script and reports whether the requested file exists. |
| `./server_check.sh` | Runs the service check script and reports whether the configured service is active. |
