# Day 16 Shell Scripting

## Scripts

- `hello.sh`: prints a greeting.
- `variables.sh`: demonstrates variables and quoting.
- `greet.sh`: reads user input.
- `check_number.sh`: checks positive, negative or zero.
- `file_check.sh`: checks if a file exists.
- `server_check.sh`: checks a service status.

## What I Learned

- Bash scripts should start with a shebang.
- Variables expand differently in single and double quotes.
- Simple conditionals are enough for useful checks.
