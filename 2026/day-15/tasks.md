Use these commands as a practical checklist for the day. Replace placeholders such as `<...>` before running them.

## Commands

| Command | What it does |
|---|---|
| `cd 2026/day-15` | Changes into the target directory before running the exercise commands. |
| `dig google.com` | Queries DNS records for the target domain. |
| `ip addr show` | Shows network interfaces and assigned IP addresses. |
| `ss -tulpn` | Lists sockets and listening services. |
| `curl -I http://myapp.com:8080` | Makes an HTTP request or checks HTTP headers for the target URL. |
| `touch day-15-networking-concepts.md` | Creates one or more empty files if they do not already exist. |
