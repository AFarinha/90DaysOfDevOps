# Day 08 Cloud Deployment

## Commands Used

Record SSH, package installation, Nginx status, HTTP check and log extraction commands.

## Challenges Faced

Document any issue with SSH keys, firewall/security groups or package installation.

## What I Learned

- Cloud instances need network rules before services are reachable.
- `systemctl` confirms whether Nginx is running.
- Nginx access and error logs provide the first troubleshooting signal.
