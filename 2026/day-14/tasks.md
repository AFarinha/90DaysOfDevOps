# Tasks - Day 14

Use these commands as a practical checklist for the day. Replace placeholders such as `<...>` before running them.

## Commands

| Command | What it does |
|---|---|
| `cd 2026/day-14` | Changes into the target directory before running the exercise commands. |
| `hostname -I` | Prints the host IP addresses or hostname information. |
| `ip addr show` | Shows network interfaces and assigned IP addresses. |
| `ping -c 4 google.com` | Tests basic network reachability to the target host. |
| `traceroute google.com` | Shows the network path to a target host. |
| `ss -tulpn` | Lists sockets and listening services. |
| `dig google.com` | Queries DNS records for the target domain. |
| `curl -I https://example.com` | Makes an HTTP request or checks HTTP headers for the target URL. |
| `netstat -an \| head` | Shows network connections and listening ports. |
| `nc -zv localhost 22` | Tests whether a TCP port is reachable. |
| `touch day-14-networking.md` | Creates one or more empty files if they do not already exist. |
