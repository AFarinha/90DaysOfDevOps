# Day 14 Networking

## Quick Concepts

- IP belongs to the Internet layer.
- TCP and UDP belong to the Transport layer.
- HTTP, HTTPS and DNS are application-level protocols.

## Hands-on Observations

Paste output snippets for IP identity, reachability, path, listening ports, DNS and HTTP checks.

## Reflection

- Fastest signal: `curl -I` for HTTP services, `ping` for basic reachability.
- If DNS fails, inspect name resolution first.
- If HTTP 500 appears, inspect application logs after network checks pass.
