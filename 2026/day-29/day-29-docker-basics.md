# Day 29 Docker Basics

## What is Docker?

Docker runs applications in containers. A container shares the host kernel but isolates filesystem, process and network context.

## Containers vs VMs

VMs virtualize a whole operating system. Containers package processes and dependencies more lightly.

## Architecture

Docker client sends commands to the Docker daemon. The daemon pulls images from registries and creates containers from those images.
