# Tasks - Day 23

Use these commands as a practical checklist for the day. Replace placeholders such as `<...>` before running them.

## Commands

| Command | What it does |
|---|---|
| `git branch` | Lists, creates or deletes Git branches depending on the flags used. |
| `git branch feature-1` | Lists, creates or deletes Git branches depending on the flags used. |
| `git switch feature-1` | Switches to another branch or creates and switches to a new branch. |
| `git switch -c feature-2` | Switches to another branch or creates and switches to a new branch. |
| `git switch main` | Switches to another branch or creates and switches to a new branch. |
| `git branch -d feature-2` | Lists, creates or deletes Git branches depending on the flags used. |
| `git remote add origin git@github.com:<user>/devops-git-practice.git` | Adds a named remote repository URL. |
| `git push -u origin main` | Uploads commits and branch references to the remote repository. |
| `git push -u origin feature-1` | Uploads commits and branch references to the remote repository. |
| `git fetch origin` | Downloads remote references without merging them into the current branch. |
| `git pull origin main` | Downloads remote changes and integrates them into the current branch. |
| `git clone https://github.com/<owner>/<repo>.git` | Creates a local copy of a remote repository. |
