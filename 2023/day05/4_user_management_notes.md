### Notes

User management in Linux is the process of creating, modifying, deleting and managing users and their permissions.

Each user has a username and a unique user ID.

The root user has full administrative permissions. Normal users have limited permissions unless they use sudo.

Common user management commands include:

- whoami: shows the current user
- id: shows user ID and group information
- useradd: creates a new user
- passwd: sets or changes a user password
- su: switches user
- sudo: runs commands with elevated permissions
